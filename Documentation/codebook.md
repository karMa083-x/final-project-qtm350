# Codebook

## Dataset Overview
This dataset contains health and demographic indicators for selected Asian countries over multiple years. Each row represents one country in a specific year.

## Variables

country  
Name of the country.  
Type: String  
Example: China  

country_code  
Three-letter ISO country code.  
Type: String  
Example: CHN  

year  
Year of observation.  
Type: Integer  
Example: 2020  

life_expectancy  
Life expectancy at birth (in years).  
Type: Numeric  

mortality_u5  
Under-five mortality rate (probability of dying before age 5, per 1,000 live births).  
Type: Numeric  

fertility_adolescent  
Adolescent fertility rate (births per 1,000 women ages 15–19).  
Type: Numeric
